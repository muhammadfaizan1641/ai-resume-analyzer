# AI Resume Analyzer

An intelligent AI-powered Resume Analyzer built with the MERN stack and Google Gemini AI to help users optimize their resumes for modern recruitment systems. The platform provides ATS scoring, resume insights, skills gap analysis, and personalized improvement recommendations through real-time AI evaluation.

---

## Features

* Resume PDF Upload and Parsing
  Upload resumes in PDF format with automated text extraction and processing.

* AI-Powered Resume Evaluation
  Uses Google Gemini 2.5 API to analyze resume content and provide intelligent feedback.

* ATS Compatibility Scoring
  Generates ATS-friendly scores to measure resume optimization for hiring systems.

* Skills Gap Analysis
  Identifies missing or weak technical and professional skills based on target roles.

* Personalized Improvement Suggestions
  Provides actionable recommendations to improve resume quality and readability.

* Real-Time AI Responses
  Delivers fast and dynamic analysis with instant API-driven feedback.

* Fully Responsive User Interface
  Optimized for desktop, tablet, and mobile devices.

* Secure Backend Architecture
  Implements secure REST APIs, file handling, and environment-based configuration.

* Drag-and-Drop Resume Upload
  Smooth and user-friendly resume upload experience.

* Analytics Dashboard
  Displays resume insights, scores, and analysis results in an organized dashboard.
