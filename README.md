AI Resume Analyzer

An AI-powered Resume Analyzer built using the MERN stack and Google Gemini AI. The application helps users evaluate resumes through ATS scoring, skills gap analysis, and actionable improvement suggestions using real-time AI feedback.

🚀 Features
📄 Resume PDF Upload and Parsing
🤖 AI-Powered Resume Analysis using Google Gemini 2.5 API
📊 ATS (Applicant Tracking System) Score Generation
🧠 Skills Gap Analysis
💡 Actionable Resume Improvement Suggestions
⚡ Real-Time API Responses
📱 Fully Responsive Mobile-Friendly UI
🔒 Secure Backend API Architecture
🎯 Drag-and-Drop Resume Upload Interface
📈 Analytics Dashboard for User Insights
